Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag DEL

Podemos marcar um texto como inserido para dar uma enfâse e indicar que ela foi adicionado depois usando a teg INS

Existe também o sublinhado com a tag U (não semântica)

Para inserir coisas do tipo X20+3, use a tag SUP
